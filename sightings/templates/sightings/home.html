Welcome to Squirrel Tracker!

You can now use this application to check 2018 central park squirrel census, add and update new squirrel data.

You can also see map that displays the locations of each squirrel sightings on OpenStreets map. If you want to want to understand more about the data, you can checkout Stats to see general statistics about 2018 central park squirrel census.

Our application used the dataset provided by NYC OpenData. To checkout more information, you can visit this website.
